INSTALACIÓN DEL ENTORNO PARA EL CURSO

1. Instalar Python 3.11 o superior
2. Instalar Visual Studio Code
3. Instalar extensiones: Python, Pylance y Jupyter
4. Crear entorno virtual
5. Instalar dependencias
6. Instalar Ollama
7. Descargar modelo llama3

Comandos básicos:

python -m venv env

Windows:
env\Scripts\activate

Mac / Linux:
source env/bin/activate

pip install requests jupyter ipykernel

Instalar Ollama desde:
https://ollama.com

Luego ejecutar:
ollama run llama3
